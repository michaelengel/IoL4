Summary

    A container for a date and time.

Methods


    type

    Returns the string "Date".


    clone

    Returns a copy of the receiver.


Comparison


    <, >, ==, !=

    All of the standard comparision methods are supported and take a Date argument.


Standard methods


    now

    Sets the receiver to the current time.


    clock

    Returns a number containing the number of seconds of processor time since the beginning of the program or -1 if unavailable.


    year

    Returns a number containing the year of the receiver.


    setYear(aNumber)

    Sets the year of the receiver.


    month

    Returns a number containing the month(1-12) of the year of the receiver.


    setMonth(aNumber)

    Sets the month of the receiver.


    day

    Returns a number containing the day of the month of the receiver.


    setDay(aNumber)

    Sets the day of the receiver.


    hour

    Returns a number containing the hour of the day(0-23) of the receiver.


    setHour(aNumber)

    Sets the hour of the receiver.


    minute

    Returns a number containing the minute of the hour(0-59) of the receiver.


    setMinute(aNumber)

    Sets the minute of the receiver.


    second

    Returns a number containing the seconds of the minute(0-59) of the receiver. This number may contain fractions of seconds.


    setSecond(aNumber)

    Sets the second of the receiver.


    isDaylightSavingsTime

    Returns self if Daylight Saving Time is in effect for the receiver, otherwise returns Nil.


    zone

    Returns a string containing the time zone code of the receiver.


Utility

    secondsSince(aDate)

    Returns a number of seconds of between aDate and the receiver.


    dateAfterSeconds(secondsNumber)

    Returns a new date that is secondsNumber seconds after the receiver.


    secondsToRun(<aMessage>)

    Returns a number containing the number of seconds taken to execute aMessage.


Conversion To


    setFormat(formatString)

    Sets the format used for the asString and print methods. Returns self. The formatString should be formatted according to ANSI C date formating rules. 

    %a abbreviated weekday name (Sun, Mon, etc.)
%A full weekday name (Sunday, Monday, etc.) 
%b abbreviated month name (Jan, Feb, etc.) 
%B full month name (January, February, etc.) 
%c full date and time string 
%d day of the month as two-digit decimal integer (01-31) 
%H hour as two-digit 24-hour clock decimal integer (00-23) 
%I hour as two-digit 12-hour clock decimal integer (01-12) 
%m month as a two-digit decimal integer (01-12)
%M minute as a two-digit decimal integer (00-59) 
%p either "AM" or "PM" 
%S second as a two-digit decimal integer (00-59) 
%U number of week in the year as two-digit decimal integer (00-52) with Sunday considered as first day of the week 
%w weekday as one-digit decimal integer (0-6) with Sunday as 0 
%W number of week in the year as two-digit decimal integer (00-52) with Monday considered as first day of the week 
%x full date string (no time); in the C locale, this is equivalent to "%m/%d/%y". 
%X full time string (no date); in the C locale, this is equivalent to "%H:%M:%S". 
%y year without century as two-digit decimal number (00-99) 
%Y year with century as four-digit decimal number 
%Z time zone name (e.g. EST); null string if no time zone can be obtained 
%% stands for '%' character in output string.

    The default format is "%Y-%m-%d %H:%M:%S %Z".


    format

    Returns a String containing the current format setting for the receiver.


    asString(optionalFormatString)

    Returns a string representation of the receiver using the receivers format. If the optionalFormatString argument is present, the receiver's format is set to it first.


    asNumber

    Returns a number representation of the receiver.


Conversion From


    fromNumber(aNumber)

    Sets the receiver to the date specified by aNumber(same format number as returned by Date asNumber). Returns self.


    fromString(aString, formatString)

    Sets the receiver to the date specified by aString as parsed according to the given formatString. See the Date asString method for formating rules. Returns self.


Math


    + aDuration

    Return a new Date with the receiver's value plus an amount of time specified by aDuration object to the receiver.


    - aDurationOrDate

    Return a new Date with the receiver's value minus an amount of time specified by aDuration to the receiver. Returns self.


    += aDuration

    Add aDuration to the receiver. Returns self.


    -= aDuration

    Subtract aDuration from the receiver. Returns self.
