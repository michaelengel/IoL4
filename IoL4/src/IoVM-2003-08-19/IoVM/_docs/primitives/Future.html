Summary

    A container for a value returned by an asynchronous message. If the value is accessed before the asynchronous message is processed, the current coroutine will pause until the value is ready.

Methods


    type

    Returns the string "Future".


    clone

    Returns a copy of the receiver.


Comparison


    <, >, ==, !=

    All of the standard comparision methods are supported and take a Future argument.


Standard methods


    value

    Returns the value of the Future, if the value is not yet ready, this will pause the current coroutine until the value is ready.


    isReady

    Returns self if the value is ready, Nil otherwise.
