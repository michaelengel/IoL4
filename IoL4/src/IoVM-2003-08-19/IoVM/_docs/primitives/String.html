Summary

    A String is a container for a byte array that can contain strings or other data. Methods that take index number arguments will treat negative indexes as the string length + the negative index. For example, the method from_to_(-1,-1) would return a string containing the last character of the receiver.

Methods

    clone

    Returns self.


    type

    Returns a string with the value "String".


Accessors


    asNumber

    Returns the receiver converted to a number. Initial whitespace is ignored.

    asString

    Returns self.


    asBuffer

    Returns a Buffer representation of the receiver.


    at(aNumber)

    Returns a number with the byte representation of the character of the receiver at the index Specified. The index is wrapped as described in the summary.


    find(aString, startIndex)

    Returns a number with the first occurrence of aString in the receiver after the startIndex. The startIndex argument is optional. By default find starts at the beginning of the string. Nil is returned if no occurrences are found.


    reverseFind(aString, startIndex)

    Returns a number with the first occurrence of aString in the receiver before the startIndex. The startIndex argument is optional. By default reverseFind starts at the end of the string. Nil is returned if no occurrences are found.


    beginsWtih(aString)

    Returns self if the receiver begins with aString, Nil otherwise.


    endsWith(aString)

    Returns self if the receiver ends with aString, Nil otherwise.


    contains(aString)

    Returns self if the receiver contains the substring aString, Nil otherwise.


    containsAnyCase(aString)

    Returns self if the receiver contains the aString regardless of casing, Nil otherwise.


    length

    Returns a number with the length of the receiver.


    split(aString1, aString2, ...)

    Returns a List of strings of the receiver split by aString1, aString2, etc. Example: 

      "this is a \ttest" split(" ", "\t")
    would return a list containing: 
      "this", "is", "a", "test"

    splitCount(aString1, aString2, ...)

    Same as split method but returns the number of results instead of a List of Strings. If you only need the count, this method is much faster than split().


    splitAt(index)

    Returns a list containing 2 strings that are the receiver split at the specified index(with the character at the index going into the 2nd string).


    substring(startIndex, endIndex)

    Returns a new string containing the substring of the receiver from the startIndex to the endIndex. The endIndex argument is optional. If not given, it is assumed to be the end of the string.


String Manipulation


    escape

    Returns a new String with escape characters replaced with escape codes. For example a string containing a single return character would contain the following 2 characters after being escaped: "\n".


    unescape

    Returns a new String with escape codes replaced with escape character.


    append(aString)

    Returns a new string with aString to the receiver.


    .. aString

    Same as append(aString).


    insert(aString, indexNumber)

    Returns a new string with aString inserted at indexNumber in the receiver.


    removeFromTo(startIndex, endIndex)

    Returns a new string that is a copy of the receiver with the substring from startIndex to endIndex removed.


    replace(aString, anotherString)

    Returns a new string with all occurances of aString replaced with anotherString in the receiver.


    lower

    Returns a copy of the receiver with all characters made lowercase.


    upper

    Returns a copy of the receiver with all characters made uppercase.


Loops


    foreach(<indexSlot>, <valueSlot>, <aMessage>)

    For each character, set indexSlot to the index of the character and valueSlot to the number containing the character value and execute aMessage. Example usage: 

      aString foreach(i, v, 
    write("byte at ", i, " is ", v, "\n")
  )
    Example use with a block: 
      myBlock = block(v, write(v, "\n")) 
  aString foreach(i, v, block(v))



Comparison


    ==, !=, >, <, >=, <=

    The String object supports all of the above standard comparison operations. Each returns self if true, Nil if false.


Utility


    print

    Prints the content of the string to standard output.


    linePrint

    Prints the content of the string to standard output followed by a new line character.


    doFile(optionalArg1, optionalArg2, ...)

    Executes the contents of the file with the path specified by the receiver. Returns the result. Any arguments are used as command line arguments would be.


    join(aList, optionalSeperatorString)

    Joins a list of strings and returns a single string result. Example: 

    l = List clone add("a", "b", "c")
result = String join(l) // result = "abc"
    If a optionalSeperatorString argument is given it is inserted between the strings in the result. Example: 
    l = List clone add("a", "b", "c")
result = String join(l, ".") // result = "a.b.c"



Base Conversions


    asBase(aNumber)

    Returns a string containing the receiver(which is assumed to be a base 10 number) converted to the specified base. Only base 8 and 16 are currently supported.


    fromBase(aNumber)

    Returns a string with a base 10 representation of the receiver converted from the specified base. Only base 8 and 16 are currently supported.


File Paths


    appendPath(aString)

    Returns a string with aString appended to the receiver such that there is one and only one path separator between the two.


    removeLastPathComponent

    Returns a string containing the receiver clipped after the last path separator.


    lastPathComponent

    Returns a string containing the receiver clipped up to the last path separator.


    pathExtension

    Returns a string containing the receiver clipped up to the last period.
