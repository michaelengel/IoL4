Summary

    A key/value dictionary appropriate for holding large key/value collections.

Methods

    clone

    Returns an new, empty Map object.


    type

    Returns the string "Map".


Accessors


    at(keyString)

    Returns the value for the key keyString. Returns nil if the key is absent.


    count

    Returns the number of key/value pairs in the receiver.


    keys

    Returns a list containing all the keys in the receiver.


    values

    Returns a list containing all the values in the receiver.


Modifiers


    atPut(keyString, aValue)

    Inserts/sets aValue with the key keyString.


    remove(keyString)

    Removes the specified keyString if present. Returns self.


    empty

    Removes all keys from the receiver.


Logical Operations


    and(aValue)

    Returns nil if aValue is nil, self otherwise.


    or(aValue)

    Returns self.


Loops


    doBlock(aBlock)

    Calls aBlock with each key and value as the first two arguments.
    Example usage: 

      aMap doBlock(block(k, v, 
    write(k, " = ", v, "\n")
  ))

    foreach(<keySlot>, <valueSlot>, <aMessage>)

    For each key value pair, sets the locals keySlot to the key and valueSlot to the value and executes aMessage.
    Example usage: 

      aMap foreach(k, v, 
    write(k, " = ", v, "\n")
  )
    Example use with a block: 
      myBlock = block(k, v, write(k, " = ", v, "\n")) 
  aMap foreach(k, v, myBlock(k, v))
    foreach() will should perform better than doBlock() unless you specifically need to execute a block.
