Summary

    A mutable, doubly linked list of values. A count is maintained so the count method is fast. Also, a pointer to the end of the list is maintained so operations on the end of the list are fast. The first index is 0.

Methods

    clone

    Returns an new LinkedList that is a shallow copy of the receiver.


    type

    Returns the string "LinkedList".


Accessors


    at(indexNumber)

    Returns the value at indexNumber.


    contains(aValue)

    Returns self if the receiver contains aValue, otherwise returns Nil.


    count

    Returns the number of items in the receiver.


    indexOf(aValue)

    Returns the index of the first occurrence of aValue. Returns Nil if no occurrence is found.


Modifiers


    add(aValue)

    Add aValue to the end of the LinkedList. Returns self.


    atPut(indexNumber, aValue)

    Puts aValue at the index specified by indexNumber. Returns aValue. If indexNumber is larger than the LinkedList size, the LinkedList is made large enough to fit it and the empty elements are set to Nil.


    empty

    Removes all items from the receiver.


    insertAt(aValue, indexNumber)

    Inserts aValue at the index specified by indexNumber. Returns aValue.


    push(aValue)

    Same as add_(aValue).


    pop

    Returns the last item in the LinkedList and removes it from the receiver. Returns nil if the receiver is empty.


    remove(aValue)

    Removes all occurances of aValue in the receiver. Returns self.


    removeAt(indexNumber)

    Removes the item at the index specified by indexNumber. Returns the value removed. Returns nil if the index is invalid.


    reverse

    Reverses the ordering of all the items in the receiver. Returns self.


Comparison


    ==(aLinkedList)

    Returns self if aLinkedList contains the same elements in the same order as the receiver, otherwise returns Nil.


Utility


    random

    Returns a random element of the receiver or Nil if the receiver is empty.


    randomize

    Randomizes the order of the elements in the receiver. Returns self.


Logical operations


    and(aValue)

    Returns nil if aValue is nil, self, otherwise.


    or(aValue)

    Returns self.


Loops


    doBlock(aBlock)

    Calls aBlock with each value as the first argument.
    Example usage: 

      aLinkedList doBlock(block(v, write(v, "\n")))

    foreach(<valueSlot>, <aMessage>)

    For each value valueSlot to the value and executes aMessage.
    Example usage: 

      aLinkedList foreach(v, 
    write(v, "\n")
  )
