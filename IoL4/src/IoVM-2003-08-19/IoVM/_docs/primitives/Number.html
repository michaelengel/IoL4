Summary

    A Number primivative is a container for a double(a large floating point number). There are two sets of number operations - ones that return new numbers(such as +, -, sin, cos) and ones that operate directly on the receiver (such as +=, -=, Sin, Cos). It's safe to operate directly on numbers without worrying about unexcepted side effects since Numbers are copied by value instead of by reference in Io. The in place operations are faster than the ones that return new Number objects. So while:

      a = a + 1
    has the same result as: 
      a ++
    The later will be much(about an order of magnitude) faster. Similarly: 
      a = a sin
    will be much slower than: 
      a Sin

Methods

    type

    Returns the string "Number".


Basic math operations


    <- aNumber

    Sets the value of the receiver to the aNumber without creating any new number objects.


    + aNumber

    Returns a new number that is the sum of the receiver and aNumber.


    += aNumber

    Sets the value of the receiver to the result of itself plus aNumber. Returns self.


    ++ aNumber

    Increments the receiver by 1. Returns self.


    - aNumber

    Returns a new number that is the difference of the receiver and aNumber.


    -= aNumber

    Sets the value of the receiver to the result of itself minus aNumber. Returns self.


    -- aNumber

    Decrements the receiver by 1. Returns self.


    / aNumber

    Returns a new number with the value of the receiver diveded by aNumber.


    /= aNumber

    Sets the value of the receiver to the result of itself divided by aNumber. Returns self.


    * aNumber

    Returns a new number that is the product of the receiver and aNumber.


    *= aNumber

    Sets the value of the receiver to the result of itself multiplied by aNumber. Returns self.


    abs

    Returns a number with the absolute value of the receiver.


    Abs

    Sets the receiver's value to the absolute value of itself. Returns self.


    exp

    Returns e to the power of the receiver.


    Exp

    Sets the receiver's value to e to the power of the receiver. Returns self.


    %(aNumber)

    Returns the receiver modulus aNumber.


    %=(aNumber)

    Sets the receiver to receiver modulus aNumber.


    Negate

    Sets the receiver's to the negative of it's current value. Returns self.


    log

    Returns the natural logarithm of the receiver.


    Log

    Sets the receiver's value to the natural logarithm of itself. Returns self.


    log10

    Returns the base 10 logarithm of the receiver.


    Log10

    Sets the receiver's value to the base 10 logarithm of itself. Returns self.


    pow(aNumber)

    Returns the value of the receiver to the aNumber power.


    Pow(aNumber)

    Sets the receiver's value to itself to the aNumber power. Returns self.


    sqrt

    Returns the square root of the receiver.


    Sqrt

    Sets the receiver's value to the square root of itself. Returns self.


Comparison


    < aNumber

    Returns the receiver if the receiver is less than aNumber and nil otherwise.


    <= aNumber

    Returns the receiver if the receiver is less than or equal to aNumber and nil otherwise.


    > aNumber

    Returns the receiver if the receiver is greater than aNumber and nil otherwise.


    >= aNumber

    Returns the receiver if the receiver is greater than or equal to aNumber and nil otherwise.


    != aNumber

    Returns the receiver if the receiver is not equal to aNumber and nil otherwise.


    == aNumber

    Returns the receiver if the receiver's value is equal to aNumber's value and nil otherwise.


    between(aNumber1, aNumber2)

    Returns the receiver if the receiver's value is between or equal to aNumber1 and aNumber2, otherwise returns nil.


Conversion


    asNumber

    Returns self.


    asString(optionalFormatString)

    Returns a string representation of the receiver. If an optionalFormatString argument is used it formats the String using the specified format and following ANSI C's scanf format conversions. For example: 

      1234.5678 asString("$.2f")
    would return: 
      $1234.56

    asCharacter

    Returns a String containing a single character whose value is the ascii value of the first byte of the receiver.


    asBuffer(optionalNumberOfBytes)

    Returns a Buffer containing a the number of bytes specified by optionalNumberOfBytes (up to the size of a double on the platform) of the reciever. If no optionalNumberOfBytes is specified, it is assumed to be the number of bytes in a double on that platform.

    roundDown

    Returns the a number with the receiver's value rounded down to the nearest integer if it's fraction component is <= .5.


    RoundDown

    Rounds the receiver's value down to the nearest integer if it's fraction component is <= .5.. Returns self.


    roundUp

    Returns the a number with the receiver's value rounded up to the nearest integer if it's fraction component is >= .5.


    RoundUp

    Rounds the receiver's value up to the nearest integer if it's fraction component is >= .5.. Returns self.


    ceil

    Returns the a number with the receiver's value rounded up to the nearest integer if it's fractional component is greater than 0.


    Ceil

    Rounds the receiver's value up to the nearest integer if it's fractional component is greater than 0. Returns self.


    floor

    Returns the a number with the receiver's value rounded down to the nearest integer if it's fractional component is not 0.


    Floor

    Rounds the receiver's value down to the nearest integer if it's fractional component is not 0.. Returns self.


    max(aNumber)

    Returns the greater of the receiver and aNumber.


    min(aNumber)

    Returns the lesser of the receiver and aNumber.

Random Number Generation


    random(optionalArg1, optionalArg2)

    If called with no arguments: returns a new floating point random Number between 0 and 1.
    If called with one argument: returns a new floating point random Number between 0 and optionalArg1.
    If called with two arguments: returns a new floating point random Number between optionalArg1 and optionalArg2.


    Random(optionalArg1, optionalArg2)

    Same as random() except the receiver is modified instead of returning a new Number.


    setRandomSeed

    Sets the random number generator seed to the value of the receiver.


Logical operations


    and(anObject)

    Returns self.


    or(anObject)

    Returns nil if anObject is nil, otherwise returns self.


    toggle

    Returns 1 if the receiver is 0. Returns 0 otherwise.


    Toggle

    If the value is not 0, sets the value of the receiver to 0. Otherwise sets the value to 1.


Bitwise operations


    Note: bitwise operations are not applied to the fractional part of numbers. The fractional bits are zeroed upon a bitwise operation.

    & aNumber

    Returns a new number with the bitwise AND of the receiver and aNumber.


    | aNumber

    Returns a new number with the bitwise OR of the receiver and aNumber.


    bitwiseAnd(aNumber)

    Applies a bitwise AND operation with aNumber to the receiver.


    bitwiseOr(aNumber)

    Applies a bitwise OR operation with aNumber to the receiver.


    bitwiseXor(aNumber)

    Applies a bitwise XOR operation with aNumber to the receiver.


    bitwiseComplement

    Turns the 0 bits of the receiver to 1s and its 1 bits to 0s.


    shiftLeft(aNumber)

    Shifts the bits of the receiver left by the number of places specified by aNumber.


    shiftRight(aNumber)

    Shifts the bits of the receiver right by the number of places specified by aNumber.
