Summary

    An mutable array of values. The first index is 0.

Methods

    clone

    Returns a new list that is a shallow copy of the receiver.


    type

    Returns the string "List".


Accessors


    at(indexNumber)

    Returns the value at indexNumber.


    contains(aValue)

    Returns self if the receiver contains aValue, otherwise returns Nil.


    count

    Returns the number of items in the receiver.


    indexOf(aValue)

    Returns the index of the first occurrence of aValue. Returns Nil if no occurrence is found.


    print(optionalSeperatorString)

    Prints the list items and returns self. The default separator string is ", ".


Modifiers


    add(aValue)

    Add aValue to the end of the list. Returns self.


    atPut(indexNumber, aValue)

    Puts aValue at the index specified by indexNumber. Returns aValue. If indexNumber is larger than the List size, the List is made large enough to fit it and the empty elements are set to Nil.


    empty

    Removes all items from the receiver.


    insertAt(aValue, indexNumber)

    Inserts aValue at the index specified by indexNumber. Returns aValue.


    push(aValue)

    Same as add(aValue).


    pop

    Returns the last item in the list and removes it from the receiver. Returns nil if the receiver is empty.


    remove(aValue)

    Removes all occurances of aValue in the receiver. Returns self.


    removeAt(indexNumber)

    Removes the item at the index specified by indexNumber. Returns the value removed. Returns nil if the index is invalid.


    reverse

    Reverses the ordering of all the items in the receiver. Returns self.


Comparison


    ==(aList)

    Returns self if aList contains the same elements in the same order as the receiver, otherwise returns Nil.


Utility


    random

    Returns a random element of the receiver or Nil if the receiver is empty.


    randomize

    Randomizes the order of the elements in the receiver. Returns self.


Logical operations


    and(aValue)

    Returns nil if aValue is nil, self, otherwise.


    or(aValue)

    Returns self.


Sorting


    sort

    Sorts items in List by internal compare method.


    sortBy(methodNameString)

    Sorts items by value returned from the specified method.


Loops


    doBlock(aBlock)

    Calls aBlock with each value as the first argument.
    Example usage: 

      aList doBlock(block(v, write(v, "\n")))

    foreach(<indexSlot>, <valueSlot>, <aMessage>)

    For each value, set the indexSlot to the index of the value and the valueSlot to the value and execute aMessage.
    Example usage: 

      aList foreach(i, v, 
    write(v, "\n")
  )
    Example use with a block: 
      myBlock = block(v, write(v, "\n")) 
  aList foreach(i, v, myBlock(v))
    foreach() will generally perform much better than doBlock() unless you specifically need to execute a block.


    select(aBlock)

    Calls aBlock with each value as the first argument and removes items that return Nil. Returns self.


    reject(aBlock)

    Calls aBlock with each value as the first argument and removes items that return non Nil. Returns self.


    detect(aBlock)

    Calls aBlock with each value as the first argument and returns the first item that return a non Nil.
