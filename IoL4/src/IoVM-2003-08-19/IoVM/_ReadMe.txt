
Io is a small object oriented programming language.

BASIC BUILD
===========

To build, type "make" in this directory. 
The executable "io" and the library "libIoVM.a" will be built.

BUILD WITH ADD ON LIBRARIES
===========================

Doing a make in the directory above this one should build any addons that came with this distribution.
